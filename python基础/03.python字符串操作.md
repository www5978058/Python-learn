## 2. 字符串下标

```python
"""
python使用2套下标
hello下标
h   e   l   l   o
0   1   2   3   4
-5  -4  -3  -2  -1
字符串长度 len(str)
"""
str = "hello"
print(str[0])
print(str[-5])
print(len(str))
```

## 3. 截取字符串切片

```python
"""
截取字符串  字符串变量[start:end],默认start为0,end默认为字符串长度
截取字符串  字符串变量[start:end:step],step为步长,-1为反着取
"""
str = "abcde"
print(str[1:2])
print(str[:3])
print(str[2:])
print(str[3::-1])
```

## 4. 查询内容

```python
"""
字符串查询内容使用find index rfind rindex count方法
find 返回第一个符合的下标，没有返回-1
rfind 右侧开始找，返回第一个符合的下标，没有返回-1
index和find区别: index找不到会报错
count 查询指定字符串数量
"""
str = "abcdefc"
print(str.find("c"))
print(str.rfind("c"))
print(str.index("c"))
print(str.rindex("c"))
print(str.count("a"))
"""
2
6
2
6
"""
```

## 5. 字符串判断

```python
"""
字符串判断 startswith endswith isalpha isdigit isalnum isspace
isalpha 是否纯字母
isdigit 是否纯数字
isalnum 是否字母或数字
isspace 是否是空白
"""
s = "hello world"
print(s.startswith("hello"))
print(s.endswith("hello"))
print("00".isdigit())
print(" ".isspace())
"""
True
False
True
True
"""
```
